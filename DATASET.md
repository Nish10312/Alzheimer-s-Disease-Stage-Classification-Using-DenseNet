# Dataset

This project uses a publicly available Alzheimer's MRI Dataset for image classification.

# Classes

- Non Demented
- Very Mild Demented
- Mild Demented
- Moderate Demented

# Download

Download the dataset from its original source (for example, Kaggle or another authorized provider).

After downloading, place it in the following directory:

dataset/
├── NonDemented/
├── VeryMildDemented/
├── MildDemented/
└── ModerateDemented/

The dataset is not included in this repository due to its size and licensing restrictions.
